<h2 id="기본-흐름">기본 흐름</h2>
<pre><code class="hljs language-mermaid">flowchart LR
  질문 --query--&gt; agent
  agent --query+context---&gt; LLM
  LLM --&gt; parser
  agent -.query.-&gt; retrievers
  retrievers -.context.-&gt; agent 
</code></pre><ul>
<li>retriever 는 도구나 검색을 추상화<ul>
<li>vector-db 를 두고 유사도 검색등이 여기서 행해진다</li>
<li>vector 화 를 하기 위해서는 사용하는 모델의 embedding 을 거쳐야한다</li>
</ul>
</li>
<li>agent 는 질문을 받아 retriever 를 사용할지 여부를 판단하고 그 결과를 <code class="span">context</code> 에 취합</li>
</ul>

</ul>

</ul>
