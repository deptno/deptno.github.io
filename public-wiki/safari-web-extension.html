safari web extension


safari web extension 은 swift 파일이 포함된 앱으로 본다
html, css, javascript 와 manifest.json 을 가지고 타겟 페이지와 통신하는 구조
manifest 에서 통신 가능한 페이지 및 새 탭에서의 사용 여부를 명시할 수 있다.
