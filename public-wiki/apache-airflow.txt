pip install apache-airflow

# 자동
# airflow standalone

# 수동
airflow db init

airflow users create \
  --username admin \
  --firstname first \
  --lastname last \
  --role Admin \
  --email deptno@gmail.com

airflow webserver --port 8080 # ui 서버
airflow scheduler # 유저가 실행할 걸 스케줄러에 할당하는 역할
```

## 실행
- web 접속 후 
- dag -> run

## 로그 확인
- web 접속 후 
- dag -> run -> event log
- dag -> run -> graph -> task -> 로그 확인
- `logs/` 디렉토리에도 생성됨
