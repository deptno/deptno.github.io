10:T8cd,<h1 id="pipenv">pipenv</h1>
<ul>
<li>virtualenv 등을 썼던 것으로 기억하는데 현재는 pipenv 로 가상환경이 추천된다한다<ul>
<li><a href="https://heytech.tistory.com/320" target="_blank">[URL] https://heytech.tistory.com/320</a></li>
</ul>
</li>
<li>프로젝트별 환경 격리를 위해 존재</li>
</ul>
<h2 id="설치">설치</h2>
<pre><code class="hljs language-sh">pip install pipenv
</code></pre><h2 id="가상환경-생성">가상환경 생성</h2>
<pre><code class="hljs language-sh"><span class="hljs-built_in">mkdir</span> project_name
<span class="hljs-built_in">cd</span> project_name
pipenv --python 3.10 <span class="hljs-comment"># 사용하고자하는버전</span>
pipenv shell <span class="hljs-comment"># 혹은 pipenv shell --python 3.10</span>
<span class="hljs-built_in">exit</span> <span class="hljs-comment"># 가상환경 나가기</span>
pipenv --<span class="hljs-built_in">rm</span> <span class="hljs-comment"># 가상환경 삭제</span>
</code></pre><blockquote><p>공식문서 보지 않고 런바이 두잉</p></blockquote><ul>
<li><code class="span">pipenv install [PACKAGE]</code> 패키지를 설치하고 <code class="span">Pipfile</code> 에 기록된다<ul>
<li><code class="span">pipenv shell</code> 활성화가 필요하진 않다</li>
<li><code class="span">pipenv install -U [PACKAGE]</code> 은 지원되지 않는 것으로 보인다</li>
</ul>
</li>
</ul>
<h3 id="pipenv-shell"><code class="span">pipenv shell</code></h3>
<ul>
<li><code class="span">shell</code> 을 활성화 한다 하더라도 <code class="span">pip</code> 를 통해 설치하면 <code class="span">Pipfile</code> 에 기록되지 않는다 다만 격리된 환경에 인스톨된다</li>
</ul>
