10:T7a3,<h1 id="grep">grep</h1>
<h2 id="옵션">옵션</h2>
<ul>
<li>o : 라인이 아닌, 매치된 단어만 표시</li>
<li>h : 헤더, 파일명등을 표시하지 않음</li>
<li>E : 정규표현식 매치 사용</li>
<li>n : 라인 표시</li>
<li>r : 특정 폴더 하위로 검색</li>
<li>v : 제외</li>
<li><code class="span">color</code> 패턴에 일치하는 경우 컬러를 부여, <code class="span">color=always</code> 와 같이 쓰는 경우 pipe 를 통해 <a href="less">less</a> 에서 컬러유지가 가능함</li>
</ul>
<h2 id="모든-라인-출력--하이라이트">모든 라인 출력 + 하이라이트</h2>
<pre><code class="hljs language-sh">grep --color -E <span class="hljs-string">&#x27;KEYWORD|$&#x27;</span> -r [DIR]
</code></pre><p><a href="ansi-color">ansi-color</a> 변경시에는 <a href="env">env</a> 설정 필요 <code class="span">GREP_COLOR</code></p>
<pre><code class="hljs language-sh">GREP_COLOR=<span class="hljs-string">&#x27;35;47&#x27;</span> grep --color -E <span class="hljs-string">&#x27;KEYWORD1|KEYWORD2|$&#x27;</span> [FILENAME]
</code></pre><blockquote><p>GREP_COLOR
It is deprecated in favor of GREP_COLORS, but still supported.
<a href="https://www.gnu.org/software/grep/manual/html_node/Environment-Variables.html" target="_blank">[URL] https://www.gnu.org/software/grep/manual/html_node/Environment-Variables.html</a></p></blockquote><p>grep 을 분리해서 여러번 파이프로 연결하는 경우 각각 다른 색상 부여도 가능함</p>
