<ul>
<li>git 을 single source of truth 로 사용, 상태가 git 과 일치될 수 있는 기능 및 UI 를 제공</li>

<li>argocd namespace 에서 application crd 를 이용해서 helm 을 매니징<ul>

</ul>
</li>

</ul>
<h2 id="install">install</h2>
<p>argocd 는 아직 helm chart 를 공식적으로 지원하지 않음, 때문에 manifest 를 통한 설치</p>
<pre><code class="hljs language-sh">kubectl create namespace argocd
kubectl apply -n argocd -f https://raw.githubusercontent.com/argoproj/argo-cd/stable/manifests/install.yaml

brew install argocd
