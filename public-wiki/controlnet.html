controlnet


개념


    

  • text -> token -> embedding 이 진행되는데 이 중간에 적용되어 최종 embedding 에 영향을 미친다
    • 

  • 이미지는 각각의 preprocessor 를 통해 필요한 이미지를 output 으로 낸다
    • 

  • text 토큰을 입력과 preprocessor 를 거친 이미지를 controlnet 모듈이 입력받아 embedding 을 형성한다
    •
