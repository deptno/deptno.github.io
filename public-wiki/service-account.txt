11:T51b,<h1 id="service-account">service-account</h1>
<ul>
<li><a href="https://kubernetes.io/docs/tasks/configure-pod-container/configure-service-account/" target="_blank">[URL] https://kubernetes.io/docs/tasks/configure-pod-container/configure-service-account/</a></li>
</ul>
<ul>
<li><a href="pod">pod</a> 가 kubernetes api 를 사용하게 위해 갖는 인증</li>
<li>user-account 와 service-account 는 구분된다</li>
<li><code class="span">automountServiceAccountToken</code> 속성을 통해서 resource 에 자동 마운트 여부를 관리할 수 있다<ul>
<li>컨테이너의 <code class="span">/var/run/secrets/kubernetes.io/serviceaccount/token</code> 에서 확인이 가능하다</li>
<li>속성을 갖는 주체는 <code class="span">ServiceAccount</code> 혹은 <code class="span">Pod</code>(우선순위) 가 된다</li>
</ul>
</li>
<li>모든 <a href="namespace">namespace</a> 는 적어도 하나(<code class="span">default</code>)의 <a href="service-account">service-account</a> 를 가진다</li>
</ul>
