10:T6ac,<h1 id="scp">scp</h1>
<p>remote &lt;-&gt; local 간의 파일 전송</p>
<pre><code class="hljs language-sh">scp -r [USER]@[HOST]:/path/to/remote/dir /path/to/local/dir
scp /path/to/local/file [USER]@[HOST]:/path/to/remote/dir
</code></pre><h2 id="options">options</h2>
<ul>
<li><code class="span">-r</code> recursive, directory 복사시 사용</li>
<li><code class="span">-p</code> preserve 수정 시간을 소스로 유지시킨다</li>
<li><code class="span">-i</code> identity file, password 대신 key file 지정</li>
<li><code class="span">-P</code> port 지정, 22 가 아닌경우 사용한다.</li>
<li><input disabled="" type="checkbox"> <code class="span">-O</code></li>
</ul>
<p><code class="span">HOST</code> 는 <code class="span">~/.ssh/config</code> 를 준수한다. 때문에 <code class="span">ssh-copy-id</code> 등을 통해 키를 설정해 두면 비번없이 편하게 복사가 가능.</p>
